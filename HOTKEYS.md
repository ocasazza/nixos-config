# Hotkey Cheatsheet

## AeroSpace (Tiling Window Manager)

### Focus

| Key     | Action      |
| ------- | ----------- |
| `opt+h` | Focus left  |
| `opt+j` | Focus down  |
| `opt+k` | Focus up    |
| `opt+l` | Focus right |

### Move Windows

| Key              | Action                        |
| ---------------- | ----------------------------- |
| `opt+shift+h`    | Move window left              |
| `opt+shift+j`    | Move window down              |
| `opt+shift+k`    | Move window up                |
| `opt+shift+l`    | Move window right             |
| `ctrl+opt+arrow` | Move window (arrow direction) |

### Resize

| Key           | Action         |
| ------------- | -------------- |
| `opt+shift+-` | Shrink by 50px |
| `opt+shift+=` | Grow by 50px   |

### Layout

| Key     | Action                           |
| ------- | -------------------------------- |
| `opt+f` | Toggle fullscreen                |
| `opt+/` | Toggle horizontal/vertical tiles |
| `opt+,` | Toggle accordion layout          |

### Workspaces

| Key             | Action                       |
| --------------- | ---------------------------- |
| `opt+1-5`       | Switch to workspace 1-5      |
| `opt+shift+1-5` | Move window to workspace 1-5 |

### Service Mode (`opt+shift+;`)

| Key         | Action                                 |
| ----------- | -------------------------------------- |
| `esc`       | Reload config and exit                 |
| `r`         | Flatten workspace tree and exit        |
| `f`         | Toggle floating/tiling and exit        |
| `backspace` | Close all windows but current and exit |

## Ghostty (Terminal)

### Splits

| Key           | Action          |
| ------------- | --------------- |
| `cmd+shift+e` | New split down  |
| `cmd+shift+o` | New split right |
| `cmd+shift+d` | Close split     |
